We’re introducing a redesigned Firezone UI to make managing secure access faster, simpler, and more intuitive.

Over time, Firezone has grown alongside the teams that rely on it. With that growth, we’ve learned a lot about how people use the product day to day as well as where things could be easier.
This update brings those improvements together in a cleaner, more streamlined experience.

### What you’ll notice

* **A simpler, more organized layout** <br />
  Find what you need faster with a clearer structure and navigation.

* **Easier access management** <br />
  Quickly understand and control who has access to what.

* **Smoother workflows** <br />
  Common tasks now take fewer steps and feel more intuitive.

### No surprises where it matters

While the interface is changing, everything under the hood stays the same:
* Your existing setup continues to work
* No changes to connectivity, Clients, or Gateways
* No disruption to your team’s access

### What to expect

We’ll begin rolling out the new UI soon, along with guides to help you get familiar with the changes.

### Built with the community

Firezone is open source, and this redesign is driven by feedback from users like you. We’re excited to keep improving it together.

Stay tuned for more updates as we roll this out to all users!
